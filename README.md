<h1>Demo JDBC - Seller Management</h1>
<p>Este é um projeto simples em Java para demonstrar a integração com bancos de dados relacionais (MySQL/MariaDB) utilizando a API JDBC (Java Database Connectivity).</p>

<p>O projeto aborda as operações básicas de persistência, tratamento de exceções SQL e o uso de PreparedStatement para segurança e performance.</p>

<h2>⚙️ Funcionalidades</h2>
<p>Conexão com banco de dados via DriverManager.

Recuperação de dados (SELECT).

Inserção de dados (INSERT).

Atualização de registros (UPDATE) com cálculos dinâmicos.

Deleção de dados (DELETE).

Fechamento seguro de recursos (Connection, Statement, ResultSet).</p>

<h2>🛠️ Tecnologias Utilizadas</h2>
<p>Java SE (JDK 17 ou superior)

MySQL / MariaDB

JDBC Driver (MySQL Connector/J)</p>
